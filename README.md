🎓 Pakistan School Education Dataset Analysis
Analysis of 10,000+ school education records across Pakistan's provinces 
using Excel and Power BI to identify regional trends and performance gaps.

📌 Project Overview
This project analyzes a public Pakistan school education dataset containing 
10,000+ records across 40 columns. The goal is to uncover regional education 
patterns, enrollment trends, and performance disparities across provinces.

🛠️ Tools & Technologies

| Category      | Tools Used              |
|---------------|-------------------------|
| Data Analysis | Microsoft Excel         |
| Visualization | Power BI                |
| Techniques    | Pivot Tables, Charts, Slicers, DAX |

🔍 Key Steps Performed
- Data cleaning and formatting in Excel
- Pivot table creation for province-wise aggregation
- Power BI report with interactive visuals and filters
- Trend analysis across enrollment, schools, and performance metrics

📊 Key Insights
- Significant enrollment gaps exist between urban and rural provinces
- Punjab has the highest number of schools but uneven distribution
- Female enrollment rate varies widely across regions
